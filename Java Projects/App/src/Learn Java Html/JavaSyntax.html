Alphabetical Java Syntax Reference

Syntax/Keyword Definition Example
abstract A non-access modifier. Can be applied to classes and methods. An abstract class cannot be instantiated, and an abstract method has no body and must be implemented by concrete subclasses. 
public abstract class Animal {
    public abstract void makeSound();
}
class Dog extends Animal {
    public void makeSound() {
        System.out.println("Woof");
    }
}
assert Used for debugging. It declares an assumption about a program state. If the assertion fails (the condition is false), an AssertionError is thrown. 
int x = 10;
assert x > 0 : "x must be positive"; // Asserts x is positive
System.out.println(x);
boolean A primitive data type that can hold one of two values: true or false. 
boolean isJavaFun = true;
boolean hasError = false;
break Used to terminate the execution of the innermost loop (for, while, do-while) or switch statement. 
for (int i = 0; i < 5; i++) {
    if (i == 3) {
        break; // Exits the loop when i is 3
    }
    System.out.println(i);
}
byte A primitive data type that stores 8-bit signed two's complement integers. Range from -128 to 127. 
byte smallNumber = 100;
case Used within a switch statement to define specific values that trigger a block of code. 
int day = 3;
switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 3:
        System.out.println("Wednesday"); // This will execute
        break;
}
catch Used in conjunction with a try block to handle exceptions that occur within the try block. 
try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Cannot divide by zero!");
}
char A primitive data type that stores a single 16-bit Unicode character. 
char initial = 'J';
char copyrightSymbol = '\u00A9';
class Used to declare a class, which is a blueprint or template for creating objects. It defines the structure and behavior of objects. 
public class Car {
    String model;
    void drive() {
        System.out.println(model + " is driving.");
    }
}
const Reserved keyword but not currently used in Java. Do not confuse with final. 
// Not used in Java
// const int MAX_VALUE = 100;
continue Used to skip the current iteration of a loop and continue with the next iteration. 
for (int i = 0; i < 5; i++) {
    if (i == 2) {
        continue; // Skips printing 2
    }
    System.out.println(i);
}
default Used in a switch statement to specify a block of code to be executed if no case matches. Also used in interfaces for default methods. 
int day = 7;
switch (day) {
    case 1: break;
    default:
        System.out.println("Weekend"); // This will execute
}
do Used to start a do-while loop, which executes its block at least once before checking the condition. 
int count = 0;
do {
    System.out.println("Count: " + count);
    count++;
} while (count < 3);
double A primitive data type that stores 64-bit double-precision floating-point numbers. Used for decimals. 
double price = 19.99;
double pi = 3.14159;
else Used with an if statement to specify a block of code to be executed if the if condition is false. 
int score = 75;
if (score >= 90) {
    System.out.println("Excellent");
} else {
    System.out.println("Good"); // This will execute
}
enum Used to declare an enumeration, which is a special type that represents a fixed set of named constants. 
public enum Day {
    SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY
}
Day today = Day.MONDAY;
extends Used in a class declaration to indicate that a class inherits from another class (superclass). This establishes an "is-a" relationship. 
class Animal { /* ... */ }
class Dog extends Animal { // Dog is an Animal
    /* ... */
}
final A non-access modifier. Can be applied to variables (makes them constants), methods (prevents overriding), and classes (prevents inheritance). 
final int MAX_SPEED = 120; // Constant variable
public final void cannotOverride() { /* ... */ }
public final class ImmutableClass { /* ... */ }
finally Used in a try-catch-finally block. The finally block always executes, regardless of whether an exception occurred or not. 
try {
    int data = 10 / 2;
} catch (ArithmeticException e) {
    System.out.println("Error occurred");
} finally {
    System.out.println("This always runs.");
}
float A primitive data type that stores 32-bit single-precision floating-point numbers. Used for decimals. 
float temperature = 98.6f; // 'f' suffix is important
float pi = 3.14f;
for Used to create a loop that iterates a specific number of times or through a collection (enhanced for loop). 
// Traditional for loop
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
// Enhanced for loop (for-each)
int[] numbers = {1, 2, 3};
for (int num : numbers) {
    System.out.println(num);
}
goto Reserved keyword but not used in Java. It is a jump statement found in some other languages. 
// Not used in Java
// goto label;
if Used for conditional execution. Executes a block of code if a specified condition is true. 
int age = 20;
if (age >= 18) {
    System.out.println("Eligible to vote");
}
implements Used in a class declaration to indicate that a class is providing the implementation for an interface. This establishes a "has-a" or "can-do" relationship. 
interface Flyable { void fly(); }
class Bird implements Flyable {
    public void fly() {
        System.out.println("Bird flying");
    }
}
import Used to bring classes, interfaces, and subpackages from other packages into the current source file, making them directly accessible without fully qualifying their names. 
import java.util.ArrayList; // Imports ArrayList class
import java.io.*; // Imports all classes in java.io package
instanceof A binary operator used to test if an object is an instance of a particular class or a subclass of that class, or implements a particular interface. 
String name = "Java";
if (name instanceof String) {
    System.out.println("name is a String");
}
int A primitive data type that stores 32-bit signed two's complement integers. Range from -2,147,483,648 to 2,147,483,647. 
int quantity = 100;
int year = 2025;
interface Used to declare an interface, which is a blueprint of a class. It can contain method signatures (implicitly abstract) and default/static methods. It defines a contract that implementing classes must adhere to. 
public interface Shape {
    double getArea();
    default void printInfo() {
        System.out.println("This is a shape.");
    }
}
long A primitive data type that stores 64-bit signed two's complement integers. Used for very large whole numbers. 
long population = 8000000000L; // 'L' suffix for long literal
native A non-access modifier that indicates a method's implementation is provided by platform-specific code (e.g., C/C++ via JNI). 
public native void doSomethingNative();
new Used to create a new instance (object) of a class or an array. It allocates memory for the new object. 
Car myCar = new Car(); // Creates a new Car object
int[] numbers = new int[5]; // Creates a new array of 5 integers
package Used to declare the package to which the classes, interfaces, and enums in the current source file belong. Helps organize classes and prevents naming conflicts. 
package com.example.myproject;
public class MyClass { /* ... */ }
private An access modifier. Members declared as private are only accessible within the class they are declared in. This promotes encapsulation. 
public class Person {
    private String name; // Only accessible within Person class
    public String getName() { return this.name; }
}
protected An access modifier. Members declared as protected are accessible within the class they are declared in, by subclasses, and by classes within the same package. 
public class Vehicle {
    protected String type; // Accessible by subclasses and same package
}
class Car extends Vehicle {
    public void displayType() {
        System.out.println(type); // Accesses protected member
    }
}
public An access modifier. Members declared as public are accessible from any other class, package, or project. It's the least restrictive access level. 
public class Main {
    public static void main(String[] args) { /* ... */ }
}
public int calculateSum(int a, int b) { /* ... */ }
return Used to exit a method. If the method has a non-void return type, it must be followed by a value of the appropriate type. 
public int add(int a, int b) {
    return a + b; // Returns the sum
}
public void printMessage() {
    System.out.println("Hello");
    return; // Optional for void methods
}
short A primitive data type that stores 16-bit signed two's complement integers. Range from -32,768 to 32,767. 
short numStudents = 15000;
static A non-access modifier. Members (variables or methods) declared as static belong to the class itself, rather than to any specific instance of the class. They can be accessed directly using the class name. 
public class MathUtils {
    public static final double PI = 3.14159; // Static constant
    public static int add(int a, int b) { // Static method
        return a + b;
    }
}
double area = MathUtils.PI * 5 * 5;
strictfp A non-access modifier used to restrict floating-point calculations to ensure strict adherence to IEEE 754 floating-point standards. 
public strictfp class Calculator {
    // Floating-point calculations inside will be strict
    public double calculate(double x, double y) {
        return x / y;
    }
}
super Refers to the immediate superclass (parent class) of the current object. Used to call a superclass's constructor or access its members. 
class Animal {
    String sound = "Generic Sound";
    Animal(String s) { this.sound = s; }
}
class Dog extends Animal {
    Dog() {
        super("Woof"); // Calls Animal's constructor
    }
    void printSound() {
        System.out.println(super.sound); // Accesses superclass member
    }
}
switch A control flow statement that allows a variable to be tested for equality against a list of values. 
int month = 4;
String season;
switch (month) {
    case 12:
    case 1:
    case 2:
        season = "Winter";
        break;
    case 3:
    case 4:
    case 5:
        season = "Spring"; // This will execute
        break;
    default:
        season = "Unknown";
}
System.out.println(season);
synchronized A non-access modifier used to control access to shared resources in multi-threaded environments. It ensures that only one thread can execute a synchronized method or block at a time for a given object. 
public class Counter {
    private int count = 0;
    public synchronized void increment() {
        count++; // Ensures atomic increment in multithreading
    }
}
this Refers to the current object. Used to distinguish instance variables from local variables/parameters with the same name, or to call another constructor in the same class. 
public class Person {
    String name;
    int age;
    public Person(String name, int age) {
        this.name = name; // Refers to the instance variable
        this.age = age;
    }
    public void display() {
        this.sayHello(); // Calls another method on the current object
    }
    private void sayHello() {
        System.out.println("Hello, " + this.name);
    }
}
throw Used to explicitly throw an exception from a method or any block of code. 
public void checkAge(int age) {
    if (age < 0) {
        throw new IllegalArgumentException("Age cannot be negative.");
    }
}
throws Used in a method signature to declare the types of exceptions that a method might throw. It's a way of telling the caller that they need to handle these exceptions. 
public void readFile(String filePath) throws IOException {
    // Code that might throw IOException
    // For example, new FileInputStream(filePath);
}
transient A non-access modifier used to mark a field of an object to indicate that it should not be serialized when the object is written to a persistent storage. 
import java.io.Serializable;
public class User implements Serializable {
    private String username;
    private transient String password; // Will not be serialized
}
try Used to define a block of code that might throw an exception. It must be followed by at least one catch block or a finally block. 
try {
    // Code that might cause an error
    int data = Integer.parseInt("abc");
} catch (NumberFormatException e) {
    System.out.println("Invalid number format!");
}
void A return type that indicates a method does not return any value. 
public void printGreeting() {
    System.out.println("Hello!");
}
volatile A non-access modifier applied to a variable. It ensures that reads and writes to this variable are atomic and always visible to all threads, preventing caching issues in multi-threaded environments. 
public class SharedResource {
    public volatile boolean flag = false; // Ensures visibility across threads
    public void toggle() {
        flag = !flag;
    }
}
while Used to create a loop that executes its block of code repeatedly as long as a specified boolean condition remains true. 
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
